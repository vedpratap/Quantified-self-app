
# How to run the code?

## VS Code
- Open folder in VS Code.
- Open **app.py** file.
- Go to **python** shell.
- Run the following commands:
  - **from app import db**
  - **db.create_all()**
  *This will create database with required tables.*
- Run **app.py** file.
- Copy the **url** and paste it in browser or press **(ctrl+link)**.
